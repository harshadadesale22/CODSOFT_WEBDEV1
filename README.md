## 📖 About

This is my personal developer portfolio — a single-page website built completely from scratch without any frameworks, showcasing my projects, technical skills, and educational journey. The site focuses on clean visual design, smooth interactions, and a fully responsive layout that works seamlessly across devices.

🔗 **Live Site:** [harshadadesale22.github.io/CODSOFT_WEBDEV1](https://harshadadesale22.github.io/CODSOFT_WEBDEV1/)

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🌓 Theme Toggle | Seamless dark / light mode switch with persistent styling |
| ⌨️ Typing Effect | Animated typing hero subtitle cycling through roles |
| 🎬 Scroll Reveal | Smooth fade/slide-in animations as sections enter the viewport |
| 📊 Scroll Progress Bar | Visual indicator of page scroll position |
| 🧊 Glassmorphism Navbar | Blurred, translucent navbar that activates on scroll |
| 🎓 Education Timeline | Expandable timeline with downloadable certificates |
| 🖼️ Certificate Modal | Click-to-zoom certificate preview |
| 📱 Fully Responsive | Optimized layout for mobile, tablet, and desktop |
| ⬆️ Back-to-Top | Smooth scroll-to-top floating button |

## ⚙️ Getting Started

Clone the repository and open it locally:

```bash
git clone https://github.com/harshadadesale22/CODSOFT_WEBDEV1.git
cd CODSOFT_WEBDEV1
```

Then simply open `index.html` in your browser, or serve it with a local development server (e.g. VS Code Live Server) for the best experience.
